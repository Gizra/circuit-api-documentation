Public Endpoints

Unauthenticated endpoints accessible without a Bearer token, intended for public-facing frontends.

These endpoints do not require authentication. Use the ?lang=en query parameter to receive translated fields in the desired language.
GET /api/v1.0/article-public

Returns a paginated list of publicly published articles, such as news, previews, and press releases.

Parameters
ParameterTypeDescription
rangeintegerNumber of results per page (default: 25)
pageintegerPage number (0-based)
sortstringField to sort by (e.g. created)
filter[status]stringFilter by article status (e.g. published)
langstringLanguage code for translated fields (e.g. en, fr)
Example Request
GET /api/v1.0/article-public?range=25&page=0&lang=en
Response
{
  "data": [
    {
      "id": "1",
      "title": "Spring Auction Preview",
      "body": "...",
      "created": "2025-09-01T00:00:00Z"
    }
  ],
  "count": 42
}

GET /api/v1.0/items-public

Returns a paginated list of publicly visible auction lots. Use filters to narrow results by sale or status.

Parameters
ParameterTypeDescription
rangeintegerNumber of results per page (default: 25)
pageintegerPage number (0-based)
sortstringField to sort by (e.g. lot_number)
filter[sale_id]stringFilter by sale node ID
filter[status]stringFilter by item status (e.g. active)
langstringLanguage code for translated fields (e.g. en, fr)
Example Request
GET /api/v1.0/items-public?filter[sale_id]=12&range=25&lang=en
Response
{
  "data": [
    {
      "id": "100",
      "title": "Baroque Oil Painting",
      "estimate_low": 5000,
      "estimate_high": 8000,
      "sale_id": "12",
      "lot_number": "15"
    }
  ],
  "count": 385
}
