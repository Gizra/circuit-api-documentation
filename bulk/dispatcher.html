Bulk Dispatcher

Dispatch bulk operations on multiple entities.

POST /bulk-dispatcher
Request Body Fields
Field Type Required Description
operation string Yes Operation to perform (e.g., print_labels)
entity_type string Yes Type of entity (item, order, client)
ids array Yes Array of entity IDs
Example Request
POST /api/v1.0/bulk-dispatcher
{
  "operation": "print_labels",
  "entity_type": "item",
  "ids": ["345", "346", "347"]
}
Response
{
  "data": {
    "success": true,
    "queue_item_id": "12345",
    "message": "Bulk operation queued successfully"
  }
}
Common Operations
  • print_labels - Print labels for items
  • print_invoices - Print invoices for orders
  • export_data - Export entity data
  • update_status - Update status for multiple entities
Note: Bulk operations are processed asynchronously. Use the queue item ID to check the status via the /advancedqueue_item endpoint.
