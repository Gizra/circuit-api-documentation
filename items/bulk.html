Items Bulk Operations

Perform bulk operations on multiple items.

POST /items-bulk
Example Request - Update Status
{
  "operation": "update_status",
  "items": ["345", "346", "347"],
  "values": {
    "status": "withdrawn"
  }
}
Example Request - Update Category
{
  "operation": "update_field",
  "items": ["345", "346"],
  "field": "category",
  "value": "15"
}
Example Request - Delete Items
{
  "operation": "delete",
  "items": ["345", "346"]
}

Items Bulk Edit (Lot Numbers)

Update lot numbers for multiple items.

PATCH /items-bulk-edit
Example Request
{
  "items": [
    {"id": "345", "lot": "1234"},
    {"id": "346", "lot": "1235"},
    {"id": "347", "lot": "1236"}
  ]
}
