Exports

Endpoints for exporting data to CSV, address lists, and third-party systems.

GET /api/v1.0/csvexport/{id}

Exports items from a sale to a third-party CSV format. Returns a CSV file download. The id path parameter is the sale node ID.

Parameters
ParameterTypeDescription
idstring (path)Sale node ID to export
Example Request
GET /api/v1.0/csvexport/45
Response
Content-Type: text/csv
Content-Disposition: attachment; filename="sale_45_export.csv"

id,title,lot_number,estimate_low,estimate_high
100,"Baroque Oil Painting",15,5000,8000
101,"Bronze Sculpture",16,3000,5000
GET /api/v1.0/address-export

Exports client address records in a structured format, suitable for mailing lists and correspondence workflows.

Parameters
ParameterTypeDescription
filter[client_id]stringFilter by client node ID
filter[country]stringFilter by country code (e.g. US, FR)
rangeintegerNumber of results per page (default: 25)
pageintegerPage number (0-based)
Example Request
GET /api/v1.0/address-export?filter[country]=US&range=100
Response
{
  "data": [
    {
      "id": "1",
      "first_name": "John",
      "last_name": "Doe",
      "address": "123 Main St",
      "city": "New York",
      "country": "US"
    }
  ],
  "count": 200
}
POST /api/v1.0/export-dynamics

Triggers an export of financial data to Microsoft Dynamics. The export is queued asynchronously and a job ID is returned for tracking.

Request Body
ParameterTypeDescription
sale_idstring (required)The sale node ID whose data should be exported
typestring (required)Export type: invoices or payments
Example Request
POST /api/v1.0/export-dynamics
{
  "sale_id": "45",
  "type": "invoices"
}
Response
{
  "data": {
    "status": "queued",
    "job_id": "dyn_export_abc123"
  }
}
POST /api/v1.0/sticker-print

Generates printable lot stickers for a set of items. Returns a URL to the generated PDF document.

Request Body
ParameterTypeDescription
item_idsarray (required)Array of item node IDs to generate stickers for
templatestring (optional)Sticker template machine name (uses default if omitted)
Example Request
POST /api/v1.0/sticker-print
{
  "item_ids": ["100", "101", "102"],
  "template": "standard_sticker"
}
Response
{
  "data": {
    "pdf_url": "/files/stickers_sale45.pdf"
  }
}
