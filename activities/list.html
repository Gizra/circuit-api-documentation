List Activities

Retrieve a list of activity records with filtering and pagination.

GET /activities
Query Parameters
Parameter Type Description Example
range integer Items per page (default: 25) ?range=50
page integer Page number (0-indexed) ?page=2
sort string Sort field and direction ?sort=-created
filter[type] string Activity type ?filter[type]=contract_signed
Example Request
curl -X GET "https://backoffice.ddev.site/api/v1.0/activities?range=10&sort=-created" -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
