Taxonomy Reference

Lookup and reference list endpoints. All taxonomy endpoints return controlled vocabulary terms used across the system.

All taxonomy endpoints support GET (list and single record). Standard query parameters: range, page, sort, filter[label].
Standard Response Shape
{
  "data": [
    {
      "id": "1",
      "label": "Value"
    }
  ],
  "count": N
}
Standard Query Parameters
ParameterTypeDescriptionExample
rangeintegerItems per page (default: 25)?range=50
pageintegerPage number (0-indexed)?page=2
sortstringSort field and direction?sort=label
filter[label]stringFilter by label?filter[label]=Painting
