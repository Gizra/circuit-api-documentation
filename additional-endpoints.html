Additional Endpoints

Reference for endpoints that do not have their own dedicated documentation pages. Endpoints covered by dedicated pages (authentication, bidding, clients, exports, files, messages, migration, orders, payments, public, users, etc.) are documented separately.

Misc Auth / Session

GET /api/v1.0/google-login-url

Returns the Google OAuth login URL for single sign-on (SSO). Use this URL to redirect users to Google for authentication.

POST /api/v1.0/get-client-token

Retrieves a client-specific token (ServerRestfulClientToken). Used to obtain a scoped token tied to a specific client record for delegated operations.

POST /api/v1.0/bid-server-auth-session

Creates a bid server authentication session. Required before performing real-time bidding operations that communicate directly with the bid server.

Endpoint Description

Sales Misc

Endpoint Description
GET /api/v1.0/sale-sessions Sale session records used for live auction sessions.
GET /api/v1.0/sale-access-type Sale access type taxonomy — defines how a sale can be accessed (public, private, by invitation, etc.).
GET /api/v1.0/sale-search Full-text sale search powered by search_api. Accepts a query string for searching across sale records.
