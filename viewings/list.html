List Viewings

Retrieve a list of viewing appointments with filtering and pagination.

GET /viewings
Query Parameters
Parameter Type Description Example
range integer Items per page (default: 25) ?range=50
page integer Page number (0-indexed) ?page=2
sort string Sort field and direction ?sort=-created
filter[item_id] string Filter by item ID ?filter[item_id]=123
Example Request
curl -X GET "https://backoffice.ddev.site/api/v1.0/viewings?range=10&sort=-created" -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
Example Response
{
  "data": [
    {
      "id": "456",
      "item_id": "789",
      "client_id": "123",
      "scheduled_date": 1640995200000,
      "status": "scheduled"
    }
  ],
  "count": 50
}
