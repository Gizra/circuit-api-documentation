List Sales (DataTable)

Retrieve a paginated list of sales/auctions.

GET /datatable-sales
Query Parameters
Parameter Type Description Example
range integer Items per page ?range=25
page integer Page number ?page=0
sort string Sort field and direction ?sort=-sale_date
filter[status] string Sale status (draft, published, closed) ?filter[status]=published
filter[sale_date][value][from] string Min sale date ?filter[sale_date][value][from]=2025-01-01
filter[sale_date][value][to] string Max sale date ?filter[sale_date][value][to]=2025-12-31
filter[catalog] string Catalog ID ?filter[catalog]=78
fields string Specific fields ?fields=id,label,status
Example Request
GET /api/v1.0/datatable-sales?filter[status]=published&sort=-sale_date&range=25
