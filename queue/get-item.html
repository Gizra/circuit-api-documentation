Get Queue Item

Get information about a queued background job.

GET /advancedqueue_item
Query Parameters
Parameter Type Description
filter[item_id] string Queue item ID
Example Request
GET /api/v1.0/advancedqueue_item?filter[item_id]=12345
Response
{
  "data": [{
    "item_id": "12345",
    "queue_id": "default",
    "state": "processing",
    "message": "Processing bulk operation",
    "num_retries": 0,
    "available": "2025-10-30T10:00:00Z",
    "processed": null
  }]
}
Queue States
  • queued - Item is waiting to be processed
  • processing - Item is currently being processed
  • success - Item was processed successfully
  • failure - Item processing failed
