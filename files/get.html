Get File

Retrieve information about an uploaded file.

GET /get-file/{fid}
Path Parameters
Parameter Type Description
fid string File ID
Example Request
GET /api/v1.0/get-file/1001
Response
{
  "data": [{
    "id": "1001",
    "url": "https://backoffice.ddev.site/sites/default/files/item-images/image.jpg",
    "filename": "image.jpg",
    "filesize": "125000"
  }]
}
Upload Files

To upload files, use:

POST /api/v1.0/files

See the API documentation for full upload details.
