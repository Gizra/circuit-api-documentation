Client Merge

Merge multiple client records into one.

POST /client-merge
Request Body
{
  "source_clients": ["456", "789"],
  "target_client": "123",
  "merge_transactions": true,
  "merge_orders": true,
  "merge_items": true
}
Response
{
  "data": {
    "merged_client_id": "123",
    "merged_count": 2,
    "deleted_clients": ["456", "789"]
  }
}
